# HTTP Status CLI

A CLI to get description about a HTTP status code

## Requirements

- [Glow](https://github.com/charmbracelet/glow)

## Installation

```bash
cargo install http-cli
```

## Usage

```bash
Usage: http-cli [OPTIONS] <STATUS_CODE>

Arguments:
  <STATUS_CODE>  

Options:
  -b, --brief    
  -h, --help     Print help
  -V, --version  Print version

```
